# Avatar

An image element with a fallback for representing the user.

## Usage

```tsx
import {
  Avatar,
  AvatarFallback,
  AvatarImage,
} from "@/registry/components/atoms/avatar"

<Avatar>
  <AvatarImage src="https://github.com/g-villarinho.png" alt="@g-villarinho" />
  <AvatarFallback>GV</AvatarFallback>
</Avatar>
```
